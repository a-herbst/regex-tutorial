--- title: Refactoring - neuer Parameter ---

In der Methode {% highlight csharp %} public virtual double Round(double value) { ... } {% endhighlight %} wurde nachträglich ein neuer Parameter hinzugefügt: {% highlight csharp %} public virtual double Round(uint numberOfDecimalPlaces, double value) { ... } {% endhighlight %} Formuliere Patterns, so dass in allen Aufrufen der Methode eine 2 als erstes Argument ergänzt wird.

Suchen nach: Ersetzen durch:
  • OtherMethod(42) + Round(23.0)
  • Console.WriteLine(Round(1.2))
  • Round(.1)

Ersetzungsergebnis:

Wenn du es richtig gemacht hast, erscheint ein Haken neben deinem regulären Ausdruck.
